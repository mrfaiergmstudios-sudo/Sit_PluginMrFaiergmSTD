# Sit_PluginMrFaiergmSTD
Плагин на команду /sit в майнкрафте для того чтобы сидеть
Плагин Находится на стадии - Alpha Возможны краши и баги.

## Особенности
- команда для того чтобы сесть - `/sit.`

## Установка
1. Скачайте файл плагина (.jar).
2. Поместите его в папку `plugins` вашего сервера.
3. Перезапустите сервер.
